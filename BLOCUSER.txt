GET
/users/list
Get Users


pour lister les utilisateurs enregistres dans le systeme.,

Returns: une liste, chaque element contenant les informations suivantes:

- **user_id** : l'identifiant de l'utilisateur nouvellement cree
- **matricule**: le matricule de l'utilisateur
- **email** : l'email de l'utilisateur
- **password** : le mot de passe de l'utilisateur qui est crypte
- **role_name** : le nom du role attribue a l'utilisateur
- **role_id** : l'identifiant du role attribue a l'utilisateur
- **status** : le status de l'utilisateur a sa creation qui est "activated"
- **timestamp** : la date de creation du nouvel utilisateur
