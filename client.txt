Clients


GET
/clients/all
Get All Client


Permet de lister tous les clients presents dans le systeme

Returns:

retourne une liste, chaque element contenant les informations suivantes:

- **client_id** : l'identifiant du client
- **nom**: le nom du client en question
- **code** : le code identifiant le client
- **sites** : la liste des sites appartenants au client. chaque site contenant le nombre le nombre d'equipements presents sur le site en question
- **timestamp** : le temps de creation de nouveau client
